Introducción

En esta primera práctica se instalaran las diferentes herramientas que se usaran durante la asignatura. El sistema operativo que usare es Windows. Los diferentes puntos de la práctica los seguiremos del guión del campus virtual. En primer lugar es darse de alta en la comunidad de Google + con mi cuenta de la universidad, seguido a esto necesitamos instalar todas las aplicaciones necesarias para poder realizar los proyectos web. Instalamos la herramienta node JS, primero descargamos el ejecutable para Windows desde la página web: http://nodejs.org/download/

También usaremos un plugin para npm, llamado express. Este lo descargamos a través de un comando desde el terminal de Windows. El comando usado es: npm install express --save
